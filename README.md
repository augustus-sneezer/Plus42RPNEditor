# Plus42 RPN Editor
A Lazarus/Free Pascal editor for HP calculator programs.

## Features

- Auto quotes  LBL "CATS"   <- LBL CATS<ENTER>  
               RCL+ "DOGS"  <- RCL+ DOGS<ENTER>  
               LBL j        <- LBL j<ENTER>
- Toggle programming mode of Plus42 from the editor
- Export to Plus42 from the editor.
- 2368*<ENTER> becomes  
  2368  
  *
- A pick-list of some RPN functions 
- Find/Replace functionality
- Import/Export from Plus42

## Building

1. Open `RPNEditor.lpi` in Lazarus
2. Build project (Ctrl+F9)
